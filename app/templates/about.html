a-view

다양한 문서 포맷을 지원하는 온라인 문서 뷰어 서비스

개요

AssetERP에서는 기존에 구글의 gview를 이용해서 Excel, Word, PowerPoint 파일을 보여주었지만, 구글이 더 이상 gview 서비스를 원활히 지원하지 않아 자체적으로 개발하게 되었습니다.

문서 변환에는 두 가지 방식이 있습니다:

  • 개별 파이썬 라이브러리를 사용하여 Excel, Word, PowerPoint를 각각 해석하는 방식
  • LibreOffice 라이브러리를 사용하는 방식 (본 프로젝트에서 채택)

본 프로젝트는 LibreOffice 라이브러리를 사용하여 안정적이고 포괄적인 문서 변환 서비스를 제공합니다.

주요 기능

Convert API

오피스 파일들(Excel, Word, PowerPoint)을 PDF 또는 HTML로 변환하여 변환된 파일의 URL을 JSON 형태로 제공합니다.

  • GET, POST 방식 지원
  • URL 또는 로컬 경로 지원
  • PDF/HTML 출력 형식 선택 가능

View API

다양한 파일 포맷을 대상으로 자동 변환하여 바로 볼 수 있는 웹페이지를 제공합니다.

  • 오피스 파일, 이미지, 텍스트, CSV, Markdown 지원
  • 파일 형식에 따른 자동 최적화
  • 반응형 웹 인터페이스

API 비교

API 대상 변환방식 제공방식
convert 오피스파일 사용자지정 JSON
view 오피스파일 외 이미지, Markdown, CSV, Text 등 대상에 따라 자동 Web Page

지원 파일 형식

파일 형식 출력 포맷 특징
오피스 파일 (Excel, Word, PPT) PDF LibreOffice를 통한 고품질 변환
텍스트 파일 (.txt) HTML UTF-8 인코딩 지원
CSV 파일 (.csv) HTML 한글 인코딩 자동 감지 (UTF-8, CP949, EUC-KR)
이미지 파일 (.jpg, .png) HTML Viewer.js를 통한 확대/축소/회전 기능
Markdown (.md) HTML Pygments 문법 강조 지원

API 사용 예제

Convert API

성공적인 변환:

$ curl "http://localhost:8003/convert?path=c:\\tmp\\sample\\11.docx&output=pdf"

{
  "success": true,
  "url": "http://127.0.0.1:8003/aview/pdf/5165f18545b0e73fd8b3e3bb69a236d8.pdf",
  "message": "로칼 파일이 OutputFormat.PDF 형식으로 변환되었습니다"
}

URL을 통한 변환:

$ curl "http://localhost:8003/convert?url=http://localhost:8003/static/files/AssetERP/1.docx&output=pdf"

{
  "success": true,
  "url": "http://127.0.0.1:8003/aview/pdf/25115ce96ff4f71d9d8c66bf7d0d74da.pdf",
  "message": "URL 문서가 OutputFormat.PDF 형식으로 변환 되었습니다"
}

View API

간단한 뷰어 호출:

http://localhost:8003/view?path=c:\\tmp\\sample\\11.docx

URL을 통한 뷰어 호출:

http://localhost:8003/view?url=http://localhost:8003/static/files/AssetERP/1.xlsx

특징:

  • 파일 형식 자동 감지
  • 최적화된 뷰어 제공
  • iframe 임베딩 지원

기술적 세부사항

환경 설정

A_VIEW_MODE 환경변수에 따라 설정이 결정됩니다:

  • development: .env 파일 사용
  • production: .env.production 파일 사용

Redis 캐싱

변환된 파일의 효율적인 관리를 위해 Redis를 사용합니다:

docker run -d --name redis-aview -p 6379:6379 redis:latest

AssetERP 연동

AssetERP 시스템에서 iframe을 통해 호출됩니다:

http://g-view-host:8003/view?url=https://.../abc.xlsx
